ESG Disclosure Analytics & MSCI Rating Prediction

NLP Machine Learning 450+ S&P 500 companies 10,000+ SEC 10-K filings 2007–2025

A full data science pipeline — from raw SEC filings to explainable ESG rating predictions — quantifying the gap between what companies say about sustainability and what they do.

1. Problem Statement

Companies produce ESG disclosures in annual 10-K filings. But are these disclosures substantive or merely rhetorical? And do rating agencies like MSCI reward rhetoric even when actual carbon emissions remain high?

This project builds a pipeline to:

  1. Score every ESG-relevant sentence in 10,000+ filings across five discourse dimensions
  2. Engineer a green hype gap metric — the mismatch between rhetoric and concrete action
  3. Predict MSCI ESG ratings out-of-sample using XGBoost
  4. Use SHAP interaction analysis to detect when rhetoric loses its legitimacy value

2. Data Collection

All S&P 500 10-K filings from 2002–2025 are downloaded from SEC EDGAR using a resumable batch pipeline with retry logic. Each submission is a full SGML/XML archive that requires stream-parsing to extract just the main document body.

download_sec_filings.py

Stream-parsing strips XML/SGML wrappers and isolates the first <DOCUMENT> block (the main 10-K text), discarding exhibits. Only the text block between <TEXT> tags is retained.

parse_sec_filings.py — extract main document
def parse_edgar_submission(filepath):
    """Stream-parse full-submission.txt; return (filed_date, main_10k_text)."""
    filed_date, text_lines = None, []
    in_first_doc = in_text_block = False
    doc_count = 0

    with open(filepath, "r", encoding="utf-8", errors="ignore") as f:
        for line in f:
            if filed_date is None and "FILED AS OF DATE:" in line:
                m = re.search(r"FILED AS OF DATE:\s*(\d{8})", line)
                if m:
                    d = m.group(1)
                    filed_date = f"{d[:4]}-{d[4:6]}-{d[6:8]}"

            stripped = line.strip()
            if stripped == "<DOCUMENT>":
                doc_count += 1
                in_first_doc = (doc_count == 1)
            elif stripped == "</DOCUMENT>" and in_first_doc:
                break          # stop after main document
            elif in_first_doc and stripped == "<TEXT>":
                in_text_block = True
            elif in_first_doc and stripped == "</TEXT>":
                in_text_block = False
            elif in_first_doc and in_text_block:
                text_lines.append(line)

    return filed_date or "UNKNOWN", "".join(text_lines)

Financial fundamentals (assets, revenue, leverage, ROA, Tobin's Q, capex intensity) are pulled from WRDS/Compustat. CO₂ emissions (Scope 1/2/3) come from the LSEG database (6.3 GB raw). MSCI ESG ratings (CCC→7 numeric scale) are merged by ticker and year.

3. NLP Scoring Pipeline

Each filing is tokenized into sentences and filtered with a keyword regex (40+ ESG-related terms). Only ESG-relevant sentences pass through to the embedding models, keeping up to 1,000 sentences per filing and capping at 50 top-scoring sentences per document.

ESG keyword pre-filter (greeness_fin_bert_model.py)
_ESG_KEYWORDS = re.compile(
    r"\b(carbon|emission|greenhouse|ghg|co2|climate|global.warm"
    r"|sustainab|environment|renewable|solar|wind.power|clean.energy"
    r"|net.zero|net zero|carbon.neutral|decarboni"
    r"|esg|waste|recycl|pollut|water.use|energy.efficien"
    r"|biodiversity|deforestation|transition.risk|physical.risk"
    r"|paris.agreement|scope.[123])\b",
    re.IGNORECASE,
)

def keyword_filter(sentences):
    return [s for s in sentences if _ESG_KEYWORDS.search(s)]

Each sentence is scored against five anchor groups using cosine similarity with a fine-tuned SentenceTransformer model. FinBERT adds a financial sentiment dimension.

Anchor groups — five ESG discourse dimensions
ALL_ANCHOR_GROUPS = {
    # How "green" the company appears (ESG branding)
    "appearance": [
        "We are committed to sustainability and environmental responsibility.",
        "The company supports climate transition and low-carbon development.",
        "We are reducing carbon emissions and improving energy efficiency.",
    ],
    # Forward-looking pledges (no current action implied)
    "commitment": [
        "We plan to reduce emissions in the future.",
        "We target net-zero and long-term environmental goals.",
    ],
    # Verifiable, measurable actions
    "concrete": [
        "We reduced emissions during the reporting period.",
        "We disclosed measurable environmental targets and implementation steps.",
        "The filing reports specific environmental metrics, targets, or expenditures.",
    ],
    # Vague / aspirational language
    "vague": [
        "We believe sustainability is important to our stakeholders.",
        "We aspire to support a greener future.",
    ],
    # Climate and regulatory risk disclosures
    "risk": [
        "Climate change may adversely affect our operations and financial condition.",
        "Transition risk may affect asset values and customer demand.",
    ],
}

embed_model = SentenceTransformer("all-MiniLM-L6-v2")
anchor_embeddings = {
    name: embed_model.encode(texts, convert_to_tensor=True, normalize_embeddings=True)
    for name, texts in ALL_ANCHOR_GROUPS.items()
}

FinBERT is applied to score each sentence for financial sentiment. Filing-level scores are computed as the top-K sentence average.

FinBERT sentiment pipeline
finbert_pipe = pipeline(
    "text-classification",
    model=AutoModelForSequenceClassification.from_pretrained("ProsusAI/finbert"),
    tokenizer=AutoTokenizer.from_pretrained("ProsusAI/finbert"),
    truncation=True, max_length=512,
)

# finbert_tone = P(positive) - P(negative) per sentence
def score_finbert(sentences):
    results = finbert_pipe(sentences, batch_size=32)
    tones = []
    for r in results:
        if r["label"] == "positive":
            tones.append(r["score"])
        elif r["label"] == "negative":
            tones.append(-r["score"])
        else:
            tones.append(0.0)
    return tones

4. Feature Engineering

The five sentence-level scores are aggregated to filing level, then two novel signals are derived: the green hype gap (rhetoric minus concrete action) and the green substance index (concrete minus vague). These capture whether a company's ESG language is backed by verifiable commitments.

Derived ESG signals
# Filing-level aggregation: mean of top-K scored sentences
filing_scores = sentence_df.groupby(["ticker", "filed_year"]).agg(
    appearance_score = ("appearance_sim", "mean"),
    concrete_score   = ("concrete_sim",   "mean"),
    vague_score      = ("vague_sim",      "mean"),
    finbert_tone     = ("finbert_tone",   "mean"),
    log_carbon_intensity = ("log_co2_intensity", "mean"),
).reset_index()

# Novel signals
filing_scores["green_hype_gap"]      = (
    filing_scores["appearance_score"] - filing_scores["concrete_score"]
)
filing_scores["green_substance_idx"] = (
    filing_scores["concrete_score"] - filing_scores["vague_score"]
)

These signals are merged with Compustat financials and LSEG CO₂ data. MSCI ESG ratings are matched by ticker and fiscal year to form the final panel.

5. Machine Learning — XGBoost Rating Predictor

An XGBoost regressor is trained to predict MSCI ESG ratings at time t using features at time t−1 (to avoid look-ahead). A strict temporal split prevents any data leakage across time periods.

Feature matrix and temporal split (xgboost_msci_model.py)
ESG_FEATURES     = ["appearance_score", "green_hype_gap", "log_carbon_intensity"]
CONTROL_FEATURES = ["log_assets", "leverage", "roa", "tobin_q", "capex_intensity"]
ALL_FEATURES     = ESG_FEATURES + CONTROL_FEATURES + ["ff12"]   # ff12 = Fama-French industry
TARGET           = "msci_rating_num"   # CCC=1 … AAA=7

# Lag-1 feature matrix: X_{t-1} → y_t
lag = df[["firm_id","year"] + ALL_FEATURES].copy()
lag["year"] = lag["year"] + 1
lag = lag.rename(columns={c: f"{c}_lag" for c in ALL_FEATURES})
ml  = df[["firm_id","year",TARGET]].merge(lag, on=["firm_id","year"], how="inner")

# Temporal split — no shuffling, no cross-validation across time
year       = ml["year"].astype(int)
train_mask = year <= 2018          # 1,413 obs
val_mask   = (year >= 2019) & (year <= 2021)  # 919 obs
test_mask  = year >= 2022          # 1,079 obs

6. SHAP Explainability & Key Finding

SHAP TreeExplainer is used for global feature importance and pairwise interaction analysis. The central question: does ESG rhetoric help ratings equally regardless of actual emissions?

SHAP interaction analysis (shap_analysis.py)
explainer  = shap.TreeExplainer(model)
shap_exp   = explainer(X_test)
shap_arr   = explainer.shap_values(X_test)

# Pairwise SHAP interaction values (n_samples × n_features × n_features)
X_test_int = X_test.copy()
for col in X_test_int.select_dtypes(["category"]).columns:
    X_test_int[col] = X_test_int[col].cat.codes   # encode categoricals first
shap_inter = explainer.shap_interaction_values(X_test_int)

# Interaction strength: ESG appearance rhetoric × carbon intensity
i_appear = feat_idx["appearance_score"]
i_carbon = feat_idx["log_carbon_intensity"]
interact  = np.abs(shap_inter[:, i_appear, i_carbon]).mean()
print(f"Mean |SHAP interaction| rhetoric × carbon: {interact:.4f}")

# Global feature importance
mean_abs_shap = pd.Series(
    np.abs(shap_arr).mean(axis=0),
    index=clean_names
).sort_values(ascending=False)
